## ✅ 已修复的问题

### 问题 1: Next.js 15 动态路由参数警告 ✅

**严重程度**: 🟡 中等（已修复）

**影响范围**:
- `blog/app/tags/[tag]/page.tsx` (3 处) ✅ 已修复
- `blog/app/posts/[slug]/page.tsx` (3 处) ✅ 已修复

**错误信息**:
```
Error: Route "/tags/[tag]" used `params.tag`. 
`params` should be awaited before using its properties.
```

**修复方案**:
```typescript
// 修复前
const tag = decodeURIComponent(params.tag); // ❌

// 修复后
const { tag } = await params; // ✅
const decodedTag = decodeURIComponent(tag);
```

**状态**: ✅ 已完成修复（2025-11-01）

**影响**:
- ❌ 控制台会显示警告
- ✅ 页面功能正常
- ✅ 不影响用户体验
- ⚠️ 不符合 Next.js 15 最佳实践
