# 🌐 浏览器测试指南

## 📋 测试前准备

1. **打开浏览器**: Chrome / Edge / Firefox
2. **访问地址**: http://localhost:3000
3. **打开开发者工具**: 按 `F12` 或 `Ctrl+Shift+I`
4. **切换到控制台标签**: Console Tab

---

## 🧪 测试步骤

### 测试 1: 首页基础功能（5分钟）

#### Step 1.1: 页面加载
```
✓ 操作: 访问 http://localhost:3000
✓ 检查: 页面是否正常显示
✓ 预期: 
  - 看到 "Webster" 大标题
  - 看到个人简介
  - 看到标签云
  - 看到文章卡片
```

#### Step 1.2: 滚动到底部查看 Footer
```
✓ 操作: 滚动到页面最底部
✓ 检查: Footer 时间信息是否显示
✓ 预期看到3个卡片:
  
  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
  │      🕐      │  │      🎂      │  │      ⏱️      │
  │   北京时间    │  │   网站创建    │  │   运行时长    │
  │   HH:mm:ss   │  │ 2025年11月1日 │  │  0天 X时 Y分 │
  └──────────────┘  └──────────────┘  └──────────────┘
  
✓ 详细检查:
  [ ] 第1个卡片: 显示北京时间（格式：14:30:45）
  [ ] 第2个卡片: 显示"2025年11月1日"
  [ ] 第3个卡片: 显示"0天 X时 Y分"
  [ ] 所有卡片有玻璃态效果（半透明、模糊）
  [ ] 图标有紫色→粉色渐变
```

#### Step 1.3: 等待观察时间更新
```
✓ 操作: 保持在 Footer 位置，观察 60 秒
✓ 检查: 北京时间是否每秒更新
✓ 预期:
  [ ] 秒数每秒增加 1
  [ ] 达到 60 秒时分钟数增加 1
  [ ] 运行时长的分钟数也会更新
```

#### Step 1.4: 测试悬停效果
```
✓ 操作: 鼠标悬停在时间卡片上
✓ 预期:
  [ ] 卡片轻微放大（scale: 1.02）
  [ ] 卡片向上浮动 4px
  [ ] 边框变为紫色渐变
  [ ] 出现阴影效果
  [ ] 动画流畅（无卡顿）
```
